# Mensageria e Whatsapp

Fluxo de mensagens ja esta implementado dentro do sistema para os usuarios que utilizam a nossa **API** do **ClickMassa** integrada ao whatsapp.
Atualmente o nosso modelo de mensagem permanece de forma fixa, ainda não permitindo a edição dessa mensagem, porem, a mensagem modelo existente realiza o preenchimento dos dados
da loja como endereço e nome do estabelecimento, em seguida informa o nome do participante atrelado a entrega e que o produto/endomenda saiu para entrega.

Futuramente será feita uma tela no **Admin** juntamente as configurações de mensageria ja existentes para que o usuario possa personalizar essa mensagem de envio.
