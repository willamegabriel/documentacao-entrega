Mensageria e Whatsapp

Fluxo de mensagens já está implementado dentro do sistema para os usuários que utilizam a nossa API do ClickMassa integrada ao WhatsApp.

Atualmente, o nosso modelo de mensagem permanece de forma fixa, ainda não permitindo a edição dessa mensagem. Porém, a mensagem modelo existente realiza o preenchimento dos dados da loja, como endereço e nome do estabelecimento, e, em seguida, informa o nome do participante atrelado à entrega e que o produto/encomenda saiu para entrega.

Futuramente será feita uma tela no Admin juntamente com as configurações de mensageria já existentes para que o usuário possa personalizar essa mensagem de envio.
